# News-Topic-Classifier-Using-BERT-
Create a model that classify the news headlines into 4 given categories

## Features
- It Classifies news headlines into:
  - World
  - Sports
  - Business
  - Sci/Tech
- Uses **Hugging Face Transformers** and **PyTorch**
- Provides a **Gradio web interface** for testing headlines easily
- Model trained on Google Colab and saved for easy use
